## Run installer for home machines
``` bash
curl -s https://raw.githubusercontent.com/yourname/dotfiles/main/bootstrap.sh | bash
```

## Curl commands for work machines
``` bash
curl -s https://raw.githubusercontent.com/yourname/dotfiles/main/bashrc -o ~/.bashrc
curl -s https://raw.githubusercontent.com/yourname/dotfiles/main/bashrc.work -o ~/.bashrc.work
```
