## Run installer for home machines
``` bash
curl -s https://raw.githubusercontent.com/chucko42/bash-dotfiles/main/bootstrap.sh | bash
```

## Curl commands for work machines
``` bash
curl -s https://raw.githubusercontent.com/chucko42/bash-dotfiles/main/bashrc -o ~/.bashrc
curl -s https://raw.githubusercontent.com/chucko42/bash-dotfiles/main/bashrc.work -o ~/.bashrc.work
```
